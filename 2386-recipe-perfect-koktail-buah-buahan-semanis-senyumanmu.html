Koktail Buah-Buahan Semanis Senyumanmu step by step

  1. Campurkan susu sejat, susu pekat dan gula di dalam periuk dan kacau sebati. Masak dia atas api guna api sederhana hingga mendidih/berbuih kecil. Tutup api dan sejukkan..
  2. Potong buah mangga dan campurkan buah-buahan dalam tin. Masukkan ketulan ais dan tuang kuah koktail. Kacau..
  3. Siap untuk di hidangkan..
