Sayur Brocoli Berasap instructions

  1. Pasang api besar dan pastikan minyak betul2 panas. Tumis bawang putih dan bawang merah..
  2. Jaga2 api ye jangan bg bawang hangus pula. Mula kecikkan api..
  3. Kemudian masukkan isi ayam dan udang..
  4. Masukkan lobak merah dan batang brocoli masak hingga sedikit layu..
  5. Selepas itu besarkan api semula dan masukkan 1/2 cawan air dr jarak agak tinggi. Masa ni api akan keluar dr bwh kuali sampai atas. So bolehla mula kecikkan api balik ye..
  6. Dan bolehla masukkan sos tiram dan kicap manis..
  7. Selepas itu masukkan brocoli dan tomato. Dan bancuhan air tepung jagung..
  8. Akhir sekali perasakan dengan garam dan gula..
  9. Siap!.
