Chap Chai instructions

  1. Bawang putih diketuk dan dicincang halus. Bawang besar dan cili api dihiris nipis.
  2. Tumis kan bawang putih bawang merah, cili api sehingga naik bau. Kemudian masukkan sedikit air. Masukkan sos tiram, kicap, garam dan serbuk perasa..
  3. Masukkan semua jenis sayur. Gaul kan seketika. Kemudian, masukkan air tepung jagung ke dalam sayur tadi..
  4. Apabila brokoli sudah lembut taburkan daun bawang, tutup kan api. Siap untuk dihidang..
