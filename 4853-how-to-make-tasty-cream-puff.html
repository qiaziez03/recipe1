Cream puff step by step

  1. Masuk kan majerin/butter bersama air ke dalam periuk dan masak sehingga cair..
  2. Kemudian, masuk kan gula, garam dan tepung gandum. Kacau sekejap dan tutup api. Kacau lagi adunan sehingga tak melekat di periuk dan kental. Ketepi kan dan biarkan suam seketika..
  3. Setelah agak lama suam, masuk kan telur satu demi satu di selangi dengan kacau perlahan. Masuk kan esen vanilla..
  4. Sekarang boleh la piping adunan ke dalam dulang. Pasti kan dulang di sapu sedikit butter dahulu utk mgelak melekat..
  5. Bakar di dalam oven lebih kurang 40minit, api atas bawah 180°darjah celcius.
  6. Utk penyediaan krim nya pula, masuk kan tepung gandum dan tepung kastard berserta sedikit susu. Kacau supaya tidak berketul dan sebati..
  7. Kemudian masuk kan telur, gula, susu segar dan esen vanilla. Kacau dengan mengguna kan api perlahan sehingga pekat dan Ketepi kan..
  8. Ambil whipping krim, kacau sehingga menjadi tanduk @likat dan campur kan ke dalam adunan tadi.kacau sehingga rata..
  9. Akhir sekali, ambil pastry dan potong di tengah jgn smpi putus, dan masuk kan krim kastard di tengah2 nya dan tabur kan dengan gula ising di atas pastry yg telah siap..
